Spacelearn
Github

You forget 90% of what you learn in a week. Use spaced learning to combat that. Create cards for every 4 study sessions and be at the top of your game!

Motivation

I was studying for the KEAM, which is the entrance exam for engineering colleges in Kerala, when I came across spaced learning.

For a couple days I manually kept track of what I learned and when I should revise it. I thought it would be a good idea to automate this and that's how SpaceLearn was born.

I learned a lot about React, Node and PosgreSQL and testing the backend API with testing tools like Chai and Mocha. I also learned how to deploy a full stack app to Heroku. Heroku shut down it's free tier a while ago so you got to wait a bit to see the app in action. :)
